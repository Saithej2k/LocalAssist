# LocalAssist CLI Baseline - 2026-07-06

Command:

```bash
DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer swift run -c release localassist-bench --iterations 100 --warmup 5 --concurrency 4 --json --output docs/performance/2026-07-06-benchmark.json
```

This benchmark measures the deterministic offline fallback path for repeatable local and CI verification. Since the 2026-07-02 baseline the engine gained the local-day due-date policy, deterministic day-name override of model dates, capture-kind inference, and new classifier verbs (pick up, text, call/pay routing). It is separate from the Xcode Instruments Foundation Models profile.

| p50 latency | 117.365 ms |
| p95 latency | 155.623 ms |

| Peak memory | 13.375 MB |
| Peak memory delta | 2.375 MB |
| Throughput | 32.451 runs/sec |
| Cancellation succeeded | true |
| Cancellation latency | 0.079 ms |

The p50 delta against 2026-07-02 (108 ms → 117 ms) reproduces on the unchanged
2026-07-02 code on the same machine on the same day, so it is machine/thermal
variance rather than an engine regression; peak memory and cancellation are
unchanged.

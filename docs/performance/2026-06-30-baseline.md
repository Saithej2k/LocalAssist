## Notes

- This baseline exercises the deterministic offline path so it is repeatable in CI and in Command Line Tools environments.
- Foundation Models live-path profiling should be captured on a supported device or simulator from full Xcode.
- Full Xcode and `simctl` were not available in this environment, so Instruments traces were not captured here.
